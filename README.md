Standard Chartered Bank Website Clone (Practice Project)

Overview:
This project is a practice-based front-end website inspired by the official Standard Chartered Bank website.
It was created purely for learning and skill improvement. The goal was to understand layout structure, responsive design, and real-world UI patterns used in banking websites.
This is not an official website and has no affiliation with Standard Chartered Bank.

Purpose of the Project:
Practice real-world website layout and design
Improve HTML, CSS, and responsive design skills
Work with structured sections like banners, services, and footers
Build confidence by cloning a professional banking website UI

Technologies Used:
HTML5
CSS3
Bootstrap
JavaScript

Key Features:
Responsive layout for desktop, tablet, and mobile
Clean and structured UI similar to a banking website
Navigation bar with multiple sections
Hero/banner section
Services and information sections
Footer with useful links

Pages / Sections:
Home
Banking Services
Cards / Accounts (UI only)
About / Information Section
Contact Section (UI only)
